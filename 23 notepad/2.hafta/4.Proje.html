WEB TASARIM DERSİ HTML NEDİR? Hyper Text Markup Languge

 

HTML işaretleme dili oalrak web sayfalarında kullanılır. Hazırlanan bu kodlar web tarayıcıları tarafından anlaşılarak bu şekilde görsel web sayfalareına dönüştürülür.HTML için programlama dili denilmez.Bu şekilde bir işlevi yoktur. HTML nedir?Diyenler için işaretleme dili olarak en kısa hali ile yanıtlamak mümkündür.HTML yazı,video veya görsel öğelerin sayfa içerisine yerleştirilmesine ve istendiği gibi konumlandırılmasını sağlar.Aynı zamanda hazırlanan sayfaların web tarayıcısında düzgün bir şekilde görülmesini sağlar. Son olarak da arama motorlarına web sayfası hakkında bilgi verilmesini sağlayan bir dildir.HTml ile web sayfası hazırlamak istiyenlerin bu durumda bilgisayarlarına program yüklemelerine gerek olmadan işlem yapma kolaylığı vardır. HTML sayfası oluşturmak için not defteri, wordpad veya word dosyası bunun için yeterli olacaktır. Bu sayede işlemler daha hızlı yapılarak istenildiği gibi kodlama yapılabilir. Hazırlanan dosyalar kayıt edilirken html, htm veya xhtml uzantısı ile kayıt edilmesi gerekir. Alıntı Kaynağı :Arma Dijital
